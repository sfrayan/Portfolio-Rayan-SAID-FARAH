Compte Rendu Micropocesseur

Cadre Cours : « SAE : Découvrir un dispositif de tranmission »
Groupe Monogroupe
Durée 1,5 semaines
Statut Terminé

Description

L'objectif de ce projet était de programmer un ESP32 avec Arduino pour lui téléverser un code permettant de transmettre des instructions en langage binaire, afin de reproduire le comportement d'un microprocesseur et d'en acquérir une meilleure compréhension.
